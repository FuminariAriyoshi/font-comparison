# Font Compare

PCにインストールされているフォント、Google Fonts、モリサワフォントを、同じテキストで一括比較できるツールです。

[English version here](README.md)

## 機能

- **ローカルフォント検出** — [`queryLocalFonts()`](https://developer.mozilla.org/ja/docs/Web/API/Window/queryLocalFonts) API でPCにインストール済みのフォントを取得（Chrome / Edge対応）
- **Google Fonts** — 人気フォント約140種を厳選収録し、必要時に読み込み
- **モリサワフォント判定** — ローカルフォントのうち、モリサワの命名規則に一致するものを自動検出（名前ベースの推測であり公式APIではありません）
- **日本語 / 英語タブ** — フォント名から自動分類し、判別できないものはCanvasに実際に描画して字形を解析（セリフの有無、等幅判定、ゴシック/明朝の線幅比較など）
- **カテゴリ絞り込み** — 英語: Sans/Mono・Serif・Script/Display・Other、日本語: ゴシック・明朝・丸ゴシック・Other
- **一覧比較グリッド** — 最大13列表示で、数百フォントも一画面でスキャン可能
- **横並び比較** — フォントカードをクリックすると比較欄に固定表示。フォント名をクリックでコピー
- **重ね合わせ比較** — Shift+クリックでフォントを重ねて表示、不透明度調整で字形の違いを直接比較
- **一括タイポグラフィ調整** — フォントサイズ・行間・字間・太さを全フォントに同時反映
- **背景の白黒切り替え**

## セットアップ

```bash
npm install
npm run dev
```

Chrome または Edge でアプリを開き、「Load local fonts」を押すとブラウザの許可ダイアログが出てローカルフォントへのアクセスが有効になります。許可しない場合はGoogle Fontsのみ表示されます。

## モリサワフォント判定について

モリサワには公開APIが存在しません。判定はフォントファミリー名を既知のモリサワ命名規則（`A-OTF`・`G-OTF`・`U-OTF`などのプレフィックス、リュウミン・新ゴ・A1明朝などの書体名）と照合する名前ベースの推測です。これはMORISAWA PASSPORT等で**既にローカルにインストール済み**のフォントにのみ有効で、未契約・未ダウンロードのフォントをプレビューすることはできません。
